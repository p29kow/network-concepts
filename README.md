## 🔑 Key Concepts Reference

### OSI Model (7 Layers)

| Layer | Name | Key Protocols/Devices |
|-------|------|----------------------|
| 7 | Application | HTTP, FTP, DNS, SMTP |
| 6 | Presentation | SSL/TLS, JPEG, ASCII |
| 5 | Session | NetBIOS, RPC |
| 4 | Transport | TCP, UDP |
| 3 | Network | IP, ICMP, Routers |
| 2 | Data Link | MAC, Ethernet, Switches |
| 1 | Physical | Cables, Hubs, Signals |

### Common Ports Cheatsheet

| Port | Protocol | Use |
|------|----------|-----|
| 21 | FTP | File Transfer |
| 22 | SSH | Secure Shell |
| 23 | Telnet | Unsecured Remote |
| 25 | SMTP | Email Sending |
| 53 | DNS | Domain Resolution |
| 80 | HTTP | Web Traffic |
| 443 | HTTPS | Secure Web Traffic |
| 3389 | RDP | Remote Desktop |
